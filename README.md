## About the App

S.C.H.E.D.U.L.E<br/>
Seek, Create, Have, Explore, Discover, Understand, Live, Experience<br/>

## Key Features 
- Collage grid Templates<br/>
- Photo or video<br/>
- Music<br/>
- Collage gallery<br/>

## Future Features
- More access to other media apps like Spotify/Tidal<br/>
- Multiple new grid styles and even a create your own grid<br/>
- More customizable themes<br/>
